# Pen Up / Pen Down

**Commands**

- **Pen Down:** `M3 S90`  
- **Pen Up:** `M3 S10` *(or `M5` on some firmware)*  
- **Dwell (wait):** `G4 P0.2` *(adjust 0.2–0.5 s if needed)*
